# Organizador de fotos

Esse projeto se trata de um organizador de fotos por pastas de data, ordenando os arquivos por ano, depois por mês, e depois por dia.

A data da foto também não é obtida pelas metainformações do arquivo, mas sim pelo nome, para evitar que arquivos copiados e colados de outros lugares da memória ainda sejam ordenados baseado na sua data real.

Realizado para uso doméstico. Se optar por usar, recomenda-se cautela: mantenha um backup dos seus arquivos em outra pasta, para evitar perda de fotos.
